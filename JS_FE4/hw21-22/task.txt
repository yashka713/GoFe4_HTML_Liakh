Если аргументы целые числа не равные нулю одновременно - возвращает наибольшийобщий делитель этих чисел, если введены 2 нуля - возвращает NaN, если хотя бы один из аргументов не есть целоечисло - возвращает undefinedfactor - принимает один аргумент. Если аргумент целое число большее 1, то возвращает массив - разложение этогочисла на простые множители (массив отсортирован в порядке возрастания, множители повторяются согласно своей кратности), иначе возвращает nullsort - принимает массив элементарных типов, возвращает этот массив, отсортированный по возрастанию (реализоватькакой-нибудь из методов, работающих за o(N log(N)) )3. Покрыть эти методы юнит-тестамиНОД(0, n) = n; НОД(m, 0) = m; НОД(m, m) = m;НОД(1, n) = 1; НОД(m, 1) = 1;Если m, n чётные, то НОД(m, n) = 2*НОД(m/2, n/2);Если m чётное, n нечётное, то НОД(m, n) = НОД(m/2, n);Если n чётное, m нечётное, то НОД(m, n) = НОД(m, n/2);Если m, n нечётные и n > m, то НОД(m, n) = НОД(m, (n-m)/2) = НОД((n-m)/2, m);Если m, n нечётные и n < m, то НОД(m, n) = НОД((m-n)/2, n);
